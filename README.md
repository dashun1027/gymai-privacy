# gymai-privacy

GymAI iOS App 的對外法律文件（Official Privacy Policy and Terms of Service for GymAI iOS App）。
此 repo **僅包含靜態 HTML**，不含任何程式碼或金鑰，可安全設為 public，並透過 GitHub Pages 託管。

## 內容
- `index.html` — 法律文件入口頁
- `privacy-policy.html` — 隱私權政策（提供給 Apple App Store / Meta 登入審查）
- `terms-of-service.html` — 服務條款

## 修訂紀錄
- **2026-06-19** — 隱私權政策 v2：補充本機儲存／可選登入、雲端同步、AI 規則運算（非外部 LLM）、相簿讀取匯入 InBody、ocrRawText、第三方登入與 ML Kit；HealthKit 標註為規劃中；刪除權利對齊「重置所有資料」。
- **2026-06-18** — 初版隱私權政策與服務條款。

### 首次啟用 Pages
1. 開啟 [Repo Settings → Pages](https://github.com/dashun1027/gymai-privacy/settings/pages)
2. **Build and deployment → Source** 選 **Deploy from a branch**
3. **Branch** 選 `main`，資料夾選 `/ (root)`，儲存
4. 約 1–2 分鐘後上述網址即可存取
