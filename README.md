# gymai-privacy

GymAI iOS App 的對外法律文件（Official Privacy Policy and Terms of Service for GymAI iOS App）。
此 repo **僅包含靜態 HTML**，不含任何程式碼或金鑰，可安全設為 public，並透過 GitHub Pages 託管。

## 內容
- `index.html` — 法律文件入口頁
- `privacy-policy.html` — 隱私權政策（提供給 Apple App Store / Meta 登入審查）
- `terms-of-service.html` — 服務條款
